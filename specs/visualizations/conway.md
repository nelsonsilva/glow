# Conway's Game of Life

Game of Life simulation with a generation trail effect.

## Visual Output

Cells evolve under standard B3/S23 rules on a toroidal grid (edges wrap). A trail of recent generations is rendered using a color palette — the newest generation gets the last palette color, older ones fade through earlier colors. On each run, a random palette is chosen from 6 options (fire, green-blue, purple-pink, earth, rainbow, ice).

Grid initializes with ~20% alive cells.

## Parameters

None.

## Frame Rate

~60fps.
